COMO USAR ESTES FICHEIROS
================================================================================

PARA DECISORES / LEGISLADORES:
  1. Ler RELATORIO_DISPOSICOES_NOVAS.md (20 min)
  2. Expandir artigos em disposicoes_novas_regulamento.html (consulta rápida)
  3. Imprimir artigos_totalmente_novos_analise_completa.docx para reuniões

PARA EQUIPAS TÉCNICAS / CONSULTORES:
  1. Carregar disposicoes_regulamento_resumido.csv em Excel
  2. Consultar disposicoes_totalmente_novas.csv para detalhes
  3. Executar disposicoes_novas_final.py para gerar dados novos

PARA COMUNICAÇÃO PÚBLICA:
  1. Abrir disposicoes_novas_regulamento.html num navegador
  2. Compartilhar link para consulta interativa
  3. Usar screenshots para apresentações

PARA ANÁLISE COMPARATIVA:
  1. Usar disposicoes_com_correspondencia_pt.csv para identificar lacunas
  2. Cruzar com disposicoes_regulamento_resumido.csv para visão completa
  3. Filtrar por tema/classificação em Excel

PARA ATUALIZAÇÕES FUTURAS:
  1. Editar dados em gerar_artigos_novos_word.py
  2. Executar python3 gerar_artigos_novos_word.py
  3. Resultado: novo documento Word automaticamente gerado
  (mesmo para HTML e CSVs com respetivos scripts)
